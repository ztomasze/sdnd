> A [[feat]]

Gain proficiency in the next level of armor. 

That is:

- If you had no armor proficiency, you can now wear Light.  
- If you had Light, you can now also wear Medium.  
- If you had Medium, you can now also wear Heavy.

| Max | Class | Prereqs |
| --- | ----- | ------- |
| 3   | Any   | None    |
